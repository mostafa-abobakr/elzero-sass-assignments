# Elzero SASS Assignments

This repository contains my solutions for the SASS (Syntactically Awesome Style Sheets) assignments from the [Elzero Web School SASS Course](https://elzero.org/category/assignments/sass-assignments/).

## 📁 Project Structure
To keep the workspace clean, every assignment block is separated into its own folder. Inside each task folder, you will typically find:
* `main.scss` - The raw SASS logic, variables, and mixins.
* `main.css` - The compiled, standard CSS output.
* `main.css.map` - The source map for debugging in the browser.

## 🚀 Concepts Covered
* SASS Variables & Scoping
* Nesting & Parent Selectors (`&`)
* Mixins (`@mixin`, `@include`) & Passing Arguments
* Inheritance & Placeholders (`@extend`, `%`)
* Built-in Math and Color Functions
* Control Flow (`@if`, `@else`, `@for`, `@each`, `@while`)
* Custom SASS Functions (`@function`, `@return`)

## 🛠️ How to View
You can view the raw logic inside the `.scss` files. If you want to experiment with the code locally, ensure you have a SASS compiler installed (like the Live Sass Compiler extension in VS Code) to re-compile the CSS.
